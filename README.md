黄金智讯·实时多维情报看板：
调用多个国际国内相关实施新闻网站数据，动态更新与黄金相关的情报线索，
包括价格动量强度 、15分钟涨跌幅 、渠道价差强度、地缘/战争热度强度、美联储/美元利率强度、美联储/利率、美国就业/失业强度、就业/失业、油价/通胀链条强度、油价/能源、科技/风险偏好强度、交易/资金流
结合实时更新的国际黄金价格和工行如意金价格，提供价格分析走势研判。
欢迎提出改进意见，都发财。

程序启动方式：
需要安装node：
（windows在cmd中一键安装node）------- winget install -e --id OpenJS.NodeJS.Lts 
（mac一键安装node）------- brew install node && node -v && npm -v
（linux一键安装node）------- sudo apt update -y && sudo apt install -y curl ca-certificates gnupg && curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash - && sudo apt install -y nodejs && node -v && npm -v

安装完成后：
windows在cmd中 node server.js 懂点代码的可以改 autostart.bat 中的路径 后双击
mac在start.command 修改路径 双击

浏览器打开：http://127.0.0.1:4173/
OK
